# project-5
Photo Gallery Web Page

A simple and responsive photo gallery web page to display multiple images in a grid layout with captions. This project is built with HTML and CSS and is perfect for beginners looking to create an elegant image gallery.

📸 Features

Displays multiple images in a neat grid layout.

Each image has a caption.

Responsive design that works on desktop and mobile devices.

Clean and minimal styling for easy customization.

🛠️ Technologies Used

HTML5

CSS3

Optional: CSS Grid or Flexbox for layout

⚡ Usage

Clone the repository:

git clone https://github.com/your-username/photo-gallery.git

Navigate to the project folder:

cd photo-gallery

Open index.html in your web browser to see the gallery.

Add or replace images in the images/ folder. Update the HTML to include new images with captions.

🎨 Customization

Modify the CSS in style.css to change colors, fonts, spacing, or the grid layout.

Add more images and captions by copying the <div class="gallery-item"> block in index.html.

Use CSS transitions to add hover effects for a modern look.

📌 Example HTML Snippet
<div class="gallery-item">
  <img src="images/photo1.jpg" alt="Photo 1">
  <p>Caption for Photo 1</p>
</div>
